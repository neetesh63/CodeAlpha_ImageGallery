# 📸 Image Gallery Pro

A modern and responsive Image Gallery built using **HTML**, **CSS**, and **JavaScript**. This project allows users to browse beautiful images from different categories with a smooth and interactive gallery experience.

---

## 🚀 Features

- 🖼️ Responsive Image Gallery
- 🚗 Vehicle Category
- 🍔 Food Category
- 🦁 Animal Category
- 🌿 Nature Category
- 🔍 Lightbox Image Preview
- ⬅️ Previous & Next Navigation
- ⌨️ Keyboard Support (← → Esc)
- 📱 Mobile Friendly Design
- ✨ Smooth Hover Animations
- 🎨 Modern Dark UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
